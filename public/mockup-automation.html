KEIN CLIP AUSGEWÄHLT — klicke einen Clip oder ziehe eine Szene aus der Palette

▸ / ▾ AUTOMATION TOGGLE

  • Kleiner Pfeil-Button direkt neben dem Track-Namen
  • Klappt eine 38px Lane unterhalb der Spur auf
  • Lane-Label zeigt Parametername mit Dropdown (VOL ▾, FILTER ▾, PAN ▾ …)
  • Indigo-Farbcodierung — eindeutig von Clip-Farben unterscheidbar

AUTOMATION CURVE

  • Stepped (Treppenform) für diskrete Werte wie Volume
  • Smooth (Kurve) für kontinuierliche Werte wie Filter-Cutoff
  • Editierbare Punkte (Dots) per Klick/Drag platzieren
  • Parameterwert orientiert sich am Mixer-Fader-Wert als Default

PARAMETER-AUSWAHL

  • Dropdown per Track: VOL · PAN · FILTER CUTOFF · REVERB · DELAY
  • Mehrere Lanes pro Track möglich (z.B. VOL + FILTER gleichzeitig)
  • Werte direkt in Mixer/AudioEngine → kein separates Routing

DATENMODELL

  • Store: arrangementAutoStore — pro Track, pro Parameter
  • Punkte: { bar: number, value: 0–1 }
  • Playback: Bei jedem Bar-Boundary → interpolierter Wert → AudioEngine
